**Билет 30. Виды соединений в языке SQL**

### Типы `JOIN` и их назначение

Пусть есть таблицы:  
- `A(id, name)`: (1, 'Алиса'), (2, 'Боб')  
- `B(id, item)`: (1, 'Книга'), (3, 'Ручка')

#### `INNER JOIN`
- **Назначение**: только совпадающие строки.
- **Результат**: (1, 'Алиса', 'Книга')
- **Синтаксис**:  
  ```sql
  SELECT * FROM A INNER JOIN B ON A.id = B.id;
  ```

#### `LEFT JOIN` (LEFT OUTER JOIN)
- **Назначение**: все строки из левой таблицы + совпадения из правой (`NULL`, если нет).
- **Результат**:  
  (1, 'Алиса', 'Книга'),  
  (2, 'Боб', NULL)
- **Синтаксис**:  
  ```sql
  SELECT * FROM A LEFT JOIN B ON A.id = B.id;
  ```

#### `RIGHT JOIN` (RIGHT OUTER JOIN)
- **Назначение**: все строки из правой таблицы + совпадения из левой.
- **Результат**:  
  (1, 'Алиса', 'Книга'),  
  (NULL, NULL, 'Ручка')
- **Синтаксис**:  
  ```sql
  SELECT * FROM A RIGHT JOIN B ON A.id = B.id;
  ```

#### `FULL OUTER JOIN`
- **Назначение**: объединение всех строк из обеих таблиц.
- **Результат**:  
  (1, 'Алиса', 'Книга'),  
  (2, 'Боб', NULL),  
  (NULL, NULL, 'Ручка')
- **Синтаксис**:  
  ```sql
  SELECT * FROM A FULL OUTER JOIN B ON A.id = B.id;
  ```

#### `CROSS JOIN`
- **Назначение**: декартово произведение (все пары).
- **Результат**: 2 × 2 = 4 строки.
- **Синтаксис**:  
  ```sql
  SELECT * FROM A CROSS JOIN B;
  ```

> ⚠️ `RIGHT JOIN` редко используется — его можно заменить `LEFT JOIN` с перестановкой таблиц.

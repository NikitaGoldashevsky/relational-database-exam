**Билет 22. Операции выборки и проекции в реляционной алгебре**

### Выборка (σ — сигма)
- **Определение**: фильтрация строк по условию.  
  Результат — подмножество кортежей, удовлетворяющих предикату.
- **Свойства**:  
  - Идемпотентна: σ<sub>P</sub>(σ<sub>P</sub>(R)) = σ<sub>P</sub>(R).  
  - Коммутативна для независимых условий: σ<sub>P1</sub>(σ<sub>P2</sub>(R)) = σ<sub>P2</sub>(σ<sub>P1</sub>(R)).
- **SQL**: `WHERE`  
  ```sql
  SELECT * FROM employees WHERE salary > 50000;
  ```

### Проекция (π — пи)
- **Определение**: выборка указанных атрибутов (столбцов) с удалением дубликатов.
- **Свойства**:  
  - Идемпотентна: π<sub>A</sub>(π<sub>A</sub>(R)) = π<sub>A</sub>(R).  
  - Не сохраняет дубликаты (в отличие от SQL без `DISTINCT`).
- **SQL**: `SELECT col1, col2` (с неявным `DISTINCT` в алгебре)  
  ```sql
  SELECT DISTINCT department, position FROM employees;
  ```

### Пример совместного применения
Отношение `Students(id, name, course, grade)`  
— Найти имена студентов, получивших «отлично»:  
  **Алгебра**: π<sub>name</sub>(σ<sub>grade = 'A'</sub>(Students))  
  **SQL**:  
  ```sql
  SELECT DISTINCT name FROM students WHERE grade = 'A';
  ```

Эти операции — основа любого запроса: сначала фильтруем (`σ`), затем выбираем нужные поля (`π`).

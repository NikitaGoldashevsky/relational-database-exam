**Билет 13. Нормальные формы 4–5**

### Четвёртая нормальная форма (4NF)
- **Требование**: отношение в BCNF + отсутствие **многозначных зависимостей**, кроме тривиальных или зависящих от суперключа.
- **Многозначная зависимость (MVD)**: X ↠ Y означает, что для каждого значения X множество значений Y не зависит от других атрибутов.
- **Нарушение**:  
  `Преподаватель(преп, курс, учебник)` — преподаватель ведёт курсы и использует учебники независимо.  
  Зависимости: `преп ↠ курс` и `преп ↠ учебник`.
- **Приведение**:
  ```sql
  CREATE TABLE преп_курсы (преп TEXT, курс TEXT);
  CREATE TABLE преп_учебники (преп TEXT, учебник TEXT);
  ```

### Пятая нормальная форма (5NF, PJ/NF)
- **Требование**: устранение **соединительных зависимостей** — когда отношение может быть восстановлено только через соединение трёх и более проекций.
- **Актуально при тернарных связях**, не сводимых к бинарным.
- **Нарушение**:  
  `Поставка(поставщик, товар, покупатель)`, где действует правило: поставщик поставляет определённые товары определённым покупателям, но комбинации не произвольны.
- **Приведение**: декомпозиция на три двумерные таблицы, если существует циклическая зависимость (редко требуется на практике).

4NF и 5NF редко применяются явно, но важны для теоретической чистоты и избежания скрытой избыточности.
